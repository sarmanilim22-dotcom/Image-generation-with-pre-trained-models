# Image Generation with Pre-trained Models

## Objective
Generate an image from a text prompt using the Stable Diffusion pre-trained model.

## Tools Used
- Google Colab
- Python
- Hugging Face
- Diffusers
- PyTorch

## Prompt
A beautiful sunset over snowy mountains, ultra realistic

## Output
The generated image is included as `generated_image.png`.
